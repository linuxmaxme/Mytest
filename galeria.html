Punto de Ventas

Facilidad y Funcionalidad es lo que le sobra a esta ventana

La ventana mas usada del sistema reúne como principales características: intuitiva, sencilla, robusta, fiable, y funcionalidades sin limites.

Sin duda es nuestra mejor carta de presentación. Desde una misma ventana podrá elegir precios mayoristas o minoristas, agregar descuentos, elegir tipos de factura, modificar números de comprobante, elegir el cliente. Si todo eso fuera poco también podrá determinar el tipo de pago. Estas son solo algunas de las características y funcionalidades mas destacadas de la misma.

Acceso al Sistema

Cada usuario tiene sus credenciales

Podemos asignar una caja y una sucursal a cada usuario. Ampliando de una manera sorprendente el control por cada usuario

Los usuarios pueden ser de una variedad muy amplia. Gestión Comercio ofrece la posibilidad de ordenarlos por grupos. Dándole los privilegios necesarios a cada grupo. Esta característica nos da una administración totalmente controlada a cada usuario. Pudiendo limitar no solo el acceso a ventanas en el sistema sino también a funcionalidades especificas de cada una.
